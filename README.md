# Free Music ArchiveThis is a repo for managing machine readable artifacts for the [Free Music Archive APIs](http://freemusicarchive.org). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Free Music Archive APIs](http://freemusicarchive.org).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://freemusicarchive.org)- [Developer](http://freemusicarchive.org)- [Twitter](https://twitter.com/freemusicarchiv)The [APIs.json](https://github.com/api-evangelist/free-music-archive/blob/master/apis.json) index provides links to all the relevant links for the Free Music Archive APIs, including the OpenAPI and Postman if they are present.
